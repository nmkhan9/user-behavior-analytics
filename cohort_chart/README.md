# 🧠 Customer Cohort Retention Analysis

This project analyzes customer retention behavior over time using **Cohort Analysis**. It is built with **Power BI**
---

## 📊 Objective

To understand **how well the business retains customers** over time by tracking groups (cohorts) of customers based on their first purchase month.

---
## 📁 Contents

- `data/` – Raw and cleaned transaction data
- `powerbi/` – Power BI `.pbix` report file
- `reports/` – Exported PDF/image versions of the dashboard
- `README.md` – Project documentation

---

## 🔍 Key Features

✅ **Cohort Analysis Table**  
Track how many users return in the following months after their first transaction.

✅ **Retention Rate Heatmap**  
Visualize how customer retention changes month-by-month across cohorts.

✅ **Dynamic Filters**  
Explore retention by time, segment, or other user attributes.

✅ **Exportable PDF Report**  
Suitable for presentations, team meetings, or decision-making.
